# Introduction

Dokumentasi endpoint API untuk backend DOMPET/Zaku.

<aside>
    <strong>Base URL</strong>: <code>http://127.0.0.1:8000</code>
</aside>

    Gunakan dokumentasi ini untuk mencoba endpoint API langsung dari browser, menyalin contoh request, atau mengunduh Postman collection dan OpenAPI spec.

    <aside>Untuk endpoint yang membutuhkan login, isi token JWT dari endpoint login pada tombol Authenticate.</aside>
